# Commands
* !blueprint (Spawn a blueprint)
* !destroy (Destroys all current jokers)
* !text (Write Funny Text On screen) (A command only available through a whitelist)
* !bankrupt (Sets money to 0)
* !jackpot (increases money by $1000)
* !inflation (Increse items in shop by $5)
* !alien (Alien Jump Scare)
* !fart (Plays a fart sound)
* !leveldown (Decrease all hand levels by 1)
* !levelup (Inscrease all hand levels by 1)
* !soul (Creates a soul spectral)
* !randomjkr (Spawn a random joker)
* !shufflejkrs (Randomize the order of current jokers)
* !shufflehand (Randomize the order of cards)
* !destroycard (Remove a random card from the players hand)
* !consumeable (Give a random Tarot, Planet or Spectral)
* !bomb (Spawn a bomb with the jokers for the player to defuse or lose the run)
* !banana (replace a random joker with a Micheal or Cavendish)
* !enhance (Give a Random Enhancement to all current cards in hand)
* !firesale (Creates a coupon tag)
* !Blind (Restricts the view with a little light.)
* !agga (Creates a glitchy effect for 30 seconds.)
